# Germinal Theory — Changelog
## Version 1.0 — 2026-06-17
### Added
- Core thesis: "Every problem has a nature—a shape, a tendency, a direction it wants to go."
- Five principles: Observe, Cultivate, Constrain, Harness, Trust.
- Anti-principles: Forcing, Controlling, Engineering (as commonly understood), Skepticism as Identity, Brittle Foundations.
- Process cycle: Encounter → Observe → Cultivate → Constrain → Harness → Trust → Step Back.
- Recognition pattern: "Like a chair."
- Cross-domain examples: Mercury Grand Marquis, M1 Garand, blacksmith as gardener, curiosity vs. skepticism.
- Discoveries section: Existing theories that resonate with Germinal Theory.
- Applications section: TRO, Aura, Analogistical Constructivism.
- Glossary and FAQ.
### Notes
- Germinal Theory was named and formalized before discovering existing theories with similar names.
- The discovery process is documented in `discoveries/convergence_notes.md`.
